ICQA

Visualització de les mesures de PM10 del ICQA per a l'Àrea Metropolitana de Barcelona al llarg de l'any 2018.

Metodologia

Interpolació amb algoritme Inverse Distance Weighting dels valors de les estacions de mesura de l'AMB obtenint una matriu de valors continua de l'extensió de l'AMB. Ajustada l'extensió geogràfica de la matriu als límits municipals, s'inverteixen els valors per obtenir un index amb rang de 0 a 200, sent 200 el pitjor dels casos i 0 el millor. De la matriu se n'extreuen isolinies d'unió de cel·les amb valors iguals i, un cop guardades en format GeoJSON, es pinten al visor tractades com a linies d'altura.

Escales

L'escala relativa utilitza els valors màxim/minim relatius a la capa per a determinar l'altura i el color de les linies representades. El rang amb el que es treballa és, per tant, variable. Els resultats d'aquest mode de visualització no són comparables entre diferents dies. L'objectiu d'aquesta escala es representar de forma inequivoca les diferències entre les diferents zones de l'AMB i servir d'eina per a l'identificació patrons de concentració relativa de contaminació.

L'escala ICQA s'ajusta al estandar del ICQA, adaptat per a la visualització. Treballa sobre un rang de 0 a 200, sent els valors de 0 a 50 molt bons, de 50.1 a 100 bons, i de 100.1 cap a amunt, dolents o molt dolents. Aquest rang es constant i per tant permet fer comparacions entre diferents moments de l'any. La distribució normal de les dades fa que els canvis i les diferències entre dies i zones quedin dissimulades.
